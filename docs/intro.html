새움터란?

'새움터'는 새로운 노동의 움직임이 시작되는 터전이라는 뜻입니다. 충남노동건강인권센터가 현장 실무자의 업무 효율을 높이고, 체계적인 노동안전보건 관리를 지원하기 위해 개발한 통합 플랫폼입니다.

엑셀 중심의 반복 업무를 디지털화하고, AI 기반 뉴스 분석으로 정책 변화에 신속히 대응할 수 있도록 돕습니다.

Purpose

왜 CNW Center인가?

산업안전보건 현장에서 겪는 세 가지 핵심 문제를 해결합니다.

📄

흩어진 데이터 통합

위험성평가, 근골조사, 상담기록이 각각 다른 엑셀 파일에 산재해 있습니다. CNW Center는 모든 업무 데이터를 하나의 시스템에서 관리합니다.

⚡

수작업 자동화

위험점수 계산, 화학물질 중대성 판정, 뉴스 수집을 자동으로 처리합니다. 반복 작업에 소비하던 시간을 현장 방문과 상담에 투자하세요.

📊

체계적 업무 관리

사업장별 조직도 기반으로 평가를 관리하고, 개선 이력을 추적합니다. 일정, 공지, 설문까지 하나의 플랫폼에서 효율적으로 운영합니다.

Features

주요 기능

노동안전보건 실무에 필요한 모든 기능을 갖추고 있습니다.

01

노동안전 뉴스 브리핑

10개 이상의 뉴스 소스에서 산업안전 관련 기사를 자동 수집하고, Claude AI가 핵심 이슈를 분석합니다. 매일 아침 '오늘의 새움터'에서 중요 뉴스를 확인할 수 있습니다.

자동 수집 AI 분석 우선순위 분류
02

위험성평가

사업장 조직도 기반으로 위험요인을 체계적으로 평가합니다. 사고, 근골격계, 화학물질, 소음 4개 카테고리별 맞춤 점수 체계를 적용하고, 개선 계획부터 완료까지 추적합니다.

5x5 매트릭스 4개 카테고리 개선이력 관리
03

근골격계유해요인조사

고용노동부 양식 기반의 4-Sheet 구조로 관리카드, 부위별 증상조사, RULA/REBA 작업자세 평가, 종합평가를 체계적으로 수행합니다.

6개 부위 평가 RULA/REBA 작업 영상 관리
04

화학물질 중대성 평가

MSDS(물질안전보건자료) 유해성 문구를 기반으로 심각성 점수를 자동 산출합니다. CAS 번호 기반 전역 데이터베이스로 사업장 간 화학물질 정보를 공유합니다.

자동 점수 산출 MSDS 연동 CAS 마스터 DB
05

상담 관리

산업재해(사고/질병/출퇴근) 상담을 체계적으로 관리합니다. 접수부터 결과 통보까지 6단계 상태 추적, 상담 이력 기록, 관련 문서 관리를 지원합니다.

6단계 상태 관리 상담 이력 문서 관리
06

설문조사

3종 기본 템플릿과 10가지 질문 유형을 지원합니다. 조건 분기 로직으로 응답에 따라 다음 질문을 동적으로 표시하고, 공개 URL로 외부 배포가 가능합니다.

템플릿 기반 조건 분기 공개 URL
07

사업장 · 일정 · 공지

사업장 등록, 5단계 조직도 관리, 드래그앤드롭 순서 변경을 지원합니다. 캘린더 기반 일정 관리와 공지사항 시스템으로 팀 협업을 돕습니다.

조직도 트리 캘린더 공지사항

Values

지향하는 가치

기술을 통해 노동안전보건의 본질적 가치를 실현합니다.

🛡️ 노동자 중심

시스템의 모든 설계는 현장 노동자의 안전과 건강을 최우선으로 합니다. 노동자의 목소리를 듣고, 위험요인을 발굴하며, 개선을 추적하는 전 과정이 노동자 보호를 향합니다.

🔧 실무 효율

현장 실무자가 반복 업무에 소비하는 시간을 줄여, 실질적인 안전 활동에 집중할 수 있도록 합니다. 자동화된 점수 계산, AI 분석, 통합 데이터 관리가 이를 실현합니다.

📈 데이터 기반 의사결정

감이 아닌 데이터로 판단합니다. 위험등급 추이, 개선율, 상담 현황을 실시간으로 파악하고, AI가 분석한 뉴스 브리핑으로 정책 변화에 선제적으로 대응합니다.

🌐 접근성과 개방성

웹 기반 시스템으로 장소와 기기에 구애받지 않습니다. 공개 설문 URL로 누구나 참여할 수 있고, QR코드를 통해 현장에서도 쉽게 접근할 수 있습니다.
