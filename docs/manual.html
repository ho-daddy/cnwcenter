새움터 사용자 매뉴얼

이 매뉴얼은 CNW Center(새움터)의 모든 기능을 상세히 안내합니다. 좌측 사이드바 또는 상단 검색을 이용해 원하는 섹션으로 이동하세요.

대상 독자
이 매뉴얼은 일반 사용자(APPROVED 상태)를 대상으로 합니다. 관리자 전용 기능(사용자 관리, 시스템 설정 등)은 별도 관리자 가이드를 참고하세요.

대시보드 — 오늘의 새움터

로그인 후 가장 먼저 보이는 화면입니다. 4개의 위젯으로 오늘 해야 할 일과 중요 정보를 한눈에 보여줍니다.

뉴스 브리핑

노동안전보건 관련 뉴스를 자동으로 수집하고, AI가 핵심 이슈를 분석하여 매일 브리핑을 제공합니다.

자동 수집 방식

시스템은 다음 5개 뉴스 소스에서 기사를 자동으로 스크래핑합니다:

소스카테고리수집 방식
매일노동뉴스노동 전문지HTML 파싱
안전신문안전 전문지HTML 파싱
세이프티퍼스트닷뉴스안전 전문지HTML 파싱
안전저널안전 전문지HTML 파싱
산업안전보건연구원정부 기관HTML 파싱

수집된 기사는 URL 해시 기반으로 중복을 제거하고, 데이터베이스에 저장됩니다.

AI 분석 결과

수집된 기사는 2단계 분석을 거칩니다:

관련성 점수 산출

Claude AI가 각 기사에 0~100점의 관련성 점수를 부여합니다. 70점 이상인 기사만 브리핑에 포함됩니다.

일일 브리핑 분석

필터링된 기사를 종합 분석하여 상위 3대 이슈를 도출합니다. 각 이슈에 대해 배경, 시사점, 실무 권고사항을 제공합니다.

브리핑 확인
매일 아침 대시보드의 '일일 브리핑' 위젯에서 오늘의 핵심 이슈를 확인하세요. 각 이슈를 클릭하면 상세 분석(배경, 시사점, 권고사항)을 볼 수 있습니다.

필터링 기준

기사는 우선순위 키워드 매칭으로 분류됩니다:

우선순위키워드 예시의미
CRITICAL사망, 중대재해, 추락사, 끼임사고즉시 확인 필요
HIGH구속, 판결, 벌금, 징역, 기소법률/사법 관련
MEDIUM산안법 개정, 위험성평가, 특별감독정책/제도 변화
LOW산업안전, 보건관리, 안전교육일반 안전보건

상담관리 시스템

산업재해(사고, 질병, 출퇴근 재해) 관련 상담을 체계적으로 관리합니다. 접수부터 결과 통보까지 전 과정을 추적합니다.

상담 등록 방법

상담관리 메뉴 진입

좌측 사이드바에서 상담관리를 클릭합니다.

새 상담 등록

새 상담 등록 버튼을 클릭하여 등록 양식을 엽니다.

노동자 정보 입력

피해 노동자의 이름, 연락처를 입력합니다. 보호자 정보는 선택사항입니다.

재해 정보 입력

사업장명, 사고유형(사고/질병/출퇴근), 사고일자, 진단명을 입력합니다. 질병의 경우 세부 카테고리(외상, 근골격계, 심혈관계, 소음성난청, 기타)를 선택합니다.

저장

등록 버튼을 클릭하면 상담 건이 생성되고, 고유 사건번호가 부여됩니다.

상담 유형 분류

유형코드설명
사고성 재해ACCIDENT작업 중 발생한 사고(추락, 끼임, 전도 등)
직업병DISEASE업무로 인한 질병(근골격계, 심혈관계, 소음성난청 등)
출퇴근 재해COMMUTE출퇴근 중 발생한 재해

질병 세부 카테고리

  • TRAUMA — 외상 (작업 중 부상)
  • MUSCULOSKELETAL — 근골격계질환
  • CARDIOVASCULAR — 심혈관질환 (과로사 등)
  • NOISE_HEARING — 소음성난청
  • OTHER — 기타 직업병

각 상담 건에서 할 수 있는 추가 작업:

  • 상담 기록 추가: 전화/방문/이메일 상담 내용과 날짜를 기록합니다.
  • 문서 관리: 진단서, 신청서, 증거자료 등을 업로드하고 관리합니다.
  • 담당자 지정: 해당 사건의 담당 스탭을 지정합니다.

위험성평가 시스템

사업장의 유해위험요인을 체계적으로 식별, 평가하고 개선 조치를 관리합니다. 조직도 기반으로 작업 단위별 평가를 수행합니다.

단계별 입력 방법

사업장 · 조직단위 선택

위험성평가 > 평가수행 메뉴에서 사업장을 선택합니다. 좌측 조직도 트리에서 평가 대상 조직단위(공정/작업)를 클릭합니다.

사전 준비
조직도가 등록되어 있어야 합니다. 사업장 관리에서 먼저 조직도를 구성하세요.

평가카드 작성

선택한 조직단위에 대해 평가카드를 생성합니다. 입력 항목:

  • 평가유형: 정기조사 또는 수시조사
  • 연도: 평가 대상 연도
  • 작업자명: 해당 공정의 대표 작업자
  • 평가자명: 평가를 수행하는 담당자
  • 근무조건: 일일근무시간, 일일생산량, 연간근무일수, 작업주기 등
  • 작업 설명: 해당 작업의 상세 내용

유해위험요인 등록

평가카드 내에서 유해위험요인을 하나씩 등록합니다. 각 요인에 대해:

  • 카테고리 선택: 사고, 근골격계, 화학물질, 소음 중 택1
  • 유해요인 기술: 구체적인 위험 요인 서술
  • 심각성 점수 (1~5): 카테고리별 기준에 따라 입력
  • 가능성 점수 (1~5): 카테고리별 기준에 따라 입력
  • 추가점수 (0~3): 카테고리별 추가 요소 체크
  • 개선계획: 해당 위험요인에 대한 개선 방안

사진 첨부

작업 현장 사진, 위험요인 사진을 업로드할 수 있습니다. 썸네일 자동 생성 및 라이트박스 보기를 지원합니다.

저장 및 확인

위험점수가 자동 계산되며, 위험등급이 색상으로 표시됩니다. 평가조회 메뉴에서 전체 평가 결과를 필터링하고 확인할 수 있습니다.

평가 기준 — 점수 체계

위험점수는 다음 공식으로 계산됩니다:

위험점수 계산 공식
위험점수 = 심각성(1~5) × 가능성(1~5) + 추가점수(0~3)
최소 1점, 최대 28점 (5 × 5 + 3)

위험등급 판정

점수 범위위험등급조치
16점 이상매우높음즉시 개선 조치, 작업 중지 검토
9~15점높음단기 개선 계획 수립, 우선 조치
5~8점보통개선 계획 수립, 주기적 점검
4점 이하낮음현 상태 유지, 정기 점검

카테고리별 평가 기준

1. 사고성 재해 (ACCIDENT)

점수심각성가능성
1부상 없음극히 드묾 (연 1회 미만, 월 10일 미만 노출)
2경미한 부상 (14일 이하 휴업)가끔 발생 (연 1회 미만, 월 10일 이상 노출)
3중등도 부상 (15~30일 휴업)때때로 발생
4중대 부상 (30일 초과 휴업)자주 발생
5사망 또는 영구장해매우 빈번 (매일 노출)

추가점수: 사고 이력 +1, 사고 우려 +1 (최대 2점)

2. 화학물질 (CHEMICAL)

점수심각성가능성 (노출빈도)
1비유해 물질연 2~3회
2경고 물질월 2~3회
3유해 물질주 2~3회
4고유해 물질일 4시간 이하
5매우 고유해 물질 (MSDS 기반)일 4시간 초과

추가점수: 관리 미흡 +1, 환기 문제 +1, 작업자 우려 +1 (최대 3점)

자동 심각성 산출
화학물질 카테고리에서 제품을 선택하면, MSDS 유해성 문구 기반으로 심각성 점수가 자동 제안됩니다.

3. 근골격계질환 (MUSCULOSKELETAL)

점수심각성 (Borg 척도)가능성 (정기)가능성 (수시)
1Borg 6~8 (매우 가벼움)연 2~3회월 20시간 미만
2Borg 9~11 (가벼움)주 1~3일, 2시간 미만월 20~50시간
3Borg 12~14 (다소 힘듦)일 4시간 이하월 50~75시간
4Borg 15~17 (힘듦)일 4~8시간월 75~100시간
5Borg 18~20 (매우 힘듦)일 8시간 초과월 100시간 초과

추가점수: 치료 이력 +1, 현재 통증 +1 (최대 2점)

4. 소음 (NOISE)

점수심각성 (소음 수준)가능성 (노출 시간)
160dB 미만일 2시간 이하
260~70dB일 4시간 이하
370~80dB일 8시간 이하
480~90dB일 8~10시간
590dB 이상일 10시간 이상

추가점수: 스트레스 +1, 청력 손실 +2 (최대 3점)

5. 절대기준 (ABSOLUTE)

특정 위험요인은 점수 계산 없이 항상 16점(매우높음)으로 평가됩니다. 즉시 개선이 필요한 비협상적 위험요인에 적용합니다.

개선이력 관리

위험요인별로 개선 계획을 수립하고 완료까지 추적합니다.

항목설명
상태계획 (PLANNED) 또는 완료 (COMPLETED)
개선일자개선 조치를 시행한 날짜
개선내용구체적인 개선 조치 사항
담당자개선 조치 책임자
개선 후 점수개선 후 심각성, 가능성, 추가점수를 재평가
사진/파일개선 전후 비교 사진, 관련 문서 첨부

근골격계유해요인조사

고용노동부 양식에 기반한 체계적인 근골격계유해요인조사를 수행합니다. 조직도와 연계하여 작업단위별로 관리합니다.

4-Sheet 구조

조사는 엑셀 양식과 동일한 4개 시트로 구성됩니다:

📋

Sheet 1: 관리카드

사업장, 작업자, 조사자 기본정보. 근무시간, 고용형태, 교대근무, 작업자율성, 유해요인 존재 여부를 기록합니다.

🧢

Sheet 2: 부위별 증상조사

6개 신체 부위별 자세 점수와 추가 요인을 평가합니다. 작업단위(ElementWork)별로 관리됩니다.

💪

Sheet 3: RULA/REBA

작업자세 평가 도구(RULA/REBA)를 적용합니다. 밀기/당기기 힘, 손/팔 자세 등을 평가합니다.

📝

Sheet 4: 종합평가

관리수준(상/중상/중/하) 판정과 종합 소견. 개선항목 목록을 관리합니다.

조사 생성

근골조사 > 조사수행에서 사업장과 조직단위를 선택하고 새 조사를 생성합니다. Sheet 1의 기본정보를 입력합니다.

작업단위 추가

조사 내에 작업단위(ElementWork)를 추가합니다. 각 작업단위별로 측정도구(중량물, 공구, 진동원 등)를 등록합니다.

부위별 평가 (Sheet 2)

작업단위별로 6개 부위를 평가합니다. 각 부위의 자세 각도를 입력하면 점수가 자동 계산됩니다.

RULA/REBA 평가 (Sheet 3)

작업자세 평가 도구를 적용합니다. 설문 형태로 답변하면 RULA/REBA 점수가 자동 산출됩니다.

종합평가 및 개선 (Sheet 4)

관리수준을 판정하고 종합 소견을 작성합니다. 개선항목을 등록하고 추적합니다.

부위별 평가 상세

6개 신체 부위 각각에 대해 자세 점수와 추가요인을 평가합니다:

부위자세 평가 요소추가 요인 (예)총점 범위

각 부위의 총점 = 자세 점수(0~5) + 추가 요인 점수이며, 최소 1점, 최대 7점으로 제한됩니다.

Sheet 3에서는 표준화된 작업자세 평가 도구를 적용합니다:

화학물질 중대성 평가

사업장에서 사용하는 화학물질의 유해성을 MSDS(물질안전보건자료) 기반으로 자동 평가합니다.

자동 판정 기준

화학물질의 심각성 점수는 MSDS에 기재된 유해성 문구를 분석하여 자동으로 산출됩니다. 다음 규칙이 적용됩니다:

점수유해성 등급해당 유해성 문구 (예)
5점 극히 위험 발암성(모든 구분), 생식독성, 변이원성(유전독성)
4점 매우 위험 급성독성 흡입 구분1, 흡인유해성, 호흡기 과민성
3점 위험 급성독성 구분2~4, 눈 손상/자극성, 표적장기독성
2점 경고 피부/눈 자극성, 피부 과민성
1점 비유해 위 항목에 해당하지 않는 물질
최고점 원칙
하나의 화학물질이 여러 유해성에 해당하면, 가장 높은 점수가 해당 물질의 심각성 점수가 됩니다. 제품의 점수는 구성 성분 중 최고 점수입니다.

점수 계산 방식

화학물질 관리는 3단계 구조로 이루어집니다:

화학 성분 (ChemicalComponent)

CAS 번호 기반의 전역 마스터 데이터베이스입니다. 성분의 유해성 문구(MSDS B02)와 규제 정보(MSDS O02)를 저장합니다. 유해성 문구에서 자동으로 심각성 점수가 산출됩니다.

화학 제품 (ChemicalProduct)

사업장에서 실제 사용하는 제품입니다. 하나의 제품은 여러 성분으로 구성되며, 제품 점수 = 구성 성분 중 최고 점수로 결정됩니다.

위험성평가 연동

화학물질 카테고리의 위험요인을 등록할 때 제품을 선택하면, 해당 제품의 심각성 점수가 자동으로 반영됩니다.

MSDS 연동

화학 성분 등록 시 MSDS 문서의 다음 항목을 입력합니다:

  • CAS 번호: 화학물질의 고유 식별 번호
  • 유해성 정보 (B02 섹션): GHS 유해성 분류 및 문구
  • 규제 정보 (O02 섹션): 해당 규제 사항

입력된 유해성 문구에서 키워드를 매칭하여 심각성 점수를 자동 산출합니다. 예를 들어 "발암성"이라는 키워드가 포함되면 자동으로 5점이 부여됩니다.

조직단위 자동 연결
화학물질 카테고리로 위험요인을 등록하면, 해당 제품이 조직단위에 자동으로 연결됩니다. 이를 통해 어떤 공정에서 어떤 화학물질을 사용하는지 추적할 수 있습니다.

사업장 관리

위험성평가와 근골조사의 기반이 되는 사업장과 조직도를 관리합니다.

사업장 등록

기본 정보 입력

사업장명, 업종, 주요 생산품/서비스, 주소, 근로자 수를 입력합니다.

담당자 등록

안전관리자, 노동조합 대표 등 주요 담당자의 이름, 직위, 연락처를 등록합니다.

조직도 구성

최대 5단계 계층 구조로 조직도를 구성합니다. 예: 부서 → 팀 → 공정 → 작업단위. 드래그앤드롭으로 순서를 변경할 수 있습니다.

조직단위 삭제 시 주의
조직단위를 삭제하면 연결된 모든 평가 데이터가 아카이브로 이동합니다. 데이터가 완전히 삭제되지는 않으며, 필요 시 복구할 수 있습니다.

설문조사

3종 기본 템플릿과 10가지 질문 유형으로 다양한 설문을 생성할 수 있습니다.

설문 생성

템플릿 선택

기본(DEFAULT), 위험성평가(RISK_ASSESSMENT), 근골조사(MUSCULOSKELETAL) 중 템플릿을 선택하거나, 빈 설문부터 시작할 수 있습니다.

질문 구성

섹션 단위로 질문을 추가합니다. 10가지 질문 유형을 지원합니다:

  • 단답형(TEXT), 숫자(NUMBER), 라디오(RADIO), 체크박스(CHECKBOX)
  • 슬라이더(RANGE), 드롭다운(DROPDOWN), 표(TABLE)
  • 순위선택(RANKED_CHOICE), 동의(CONSENT)

조건 분기 설정

이전 답변에 따라 질문을 표시/숨김 처리할 수 있습니다. AND/OR 조합으로 복잡한 조건도 구현 가능합니다.

배포

설문을 게시하면 고유한 공개 URL이 생성됩니다. 이 URL을 공유하거나 QR코드로 변환하여 배포합니다. 인증 없이 누구나 응답할 수 있습니다.

응답 분석

수집된 응답은 질문별 분석과 함께 CSV/Excel로 내보낼 수 있습니다. 설문 복제 기능으로 기존 설문을 다른 사업장이나 연도에 재활용할 수 있습니다.
