02

RISK ASSESSMENT

위험성평가

조직도 기반 체계적 평가, 4개 카테고리

위험성평가 수행 단계

5단계 평가 프로세스

사업장 · 조직단위 선택

조직도 트리에서 평가 대상 공정/작업을 선택합니다

평가카드 작성

평가유형(정기/수시), 작업자명, 근무조건 입력

유해위험요인 등록

카테고리별(사고/근골/화학/소음) 위험요인과 점수 입력

사진 첨부

현장 사진, 위험요인 사진 업로드 (라이트박스 지원)

결과 확인 · 개선 등록

자동 산출된 위험등급 확인, 개선 계획 수립

점수 체계

위험점수 계산

위험점수 = 심각성(1~5) × 가능성(1~5)
+ 추가점수(0~3)

최대 28점 (5×5+3)

4개 카테고리: 사고, 근골격계, 화학물질, 소음 — 각각 심각성/가능성 기준이 다릅니다.

위험등급

점수등급조치
16+매우높음즉시 개선, 작업 중지
9~15높음단기 개선 계획
5~8보통주기적 점검
~4낮음현 상태 유지

개선이력: 계획(PLANNED) → 완료(COMPLETED), 개선 전후 점수 비교, 사진/파일 첨부 가능

03

MUSCULOSKELETAL

근골격계유해요인조사

4-Sheet 구조, 6개 부위, RULA/REBA

4-Sheet 구조

조사 시트 구성

Sheet 1: 관리카드

사업장, 작업자, 조사자 기본정보
근무시간, 고용형태, 교대근무
유해요인 존재 여부 체크

Sheet 2: 부위별 평가

6개 부위 자세 각도 입력
추가 요인(도구, 진동 등) 체크
총점 자동 산출 (1~7점)

Sheet 3: RULA/REBA

표준 작업자세 평가 도구
설문 답변 → 점수 자동 산출
밀기/당기기 힘 평가

Sheet 4: 종합평가

관리수준 판정 (상/중상/중/하)
종합 소견 작성
개선항목 등록 · 추적

부위별 평가

6개 신체 부위 점수 체계

부위자세 평가추가 요인 예시범위
손/손목굴곡/신전, 내전/외전공구 중량, 진동, 손가락 집기1~7
팔꿈치굴곡, 회내/회외공구 중량, 저온, 손목 편향1~7
어깨/위팔외전, 굴곡, 거상공구 중량, 반복, 팔 지지1~7
목굴곡/신전, 회전시각 부담, 압력, 힘 발휘1~7
허리굴곡, 회전, 좌식/입식중량물, 비틀림, 진동1~7
무릎/발목무릎 꿇기, 쪼그리기장시간 서기, 사다리, 저온1~7

총점 = 자세 점수(0~5) + 추가 요인 점수, 최소 1점 ~ 최대 7점
