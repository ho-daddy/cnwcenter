문제

현장의 어려움

  • 위험성평가, 근골조사, 상담기록이 각각 다른 엑셀 파일에 산재
  • 위험점수, 화학물질 심각성을 수작업으로 계산
  • 노동안전 뉴스와 정책 변화를 개별적으로 추적
  • 사업장별 조직 현황 파악과 이력 추적이 어려움

솔루션

새움터가 제공하는 가치

  • 통합 데이터 관리 — 모든 업무 데이터를 하나의 시스템에서
  • 자동화 — 위험점수, 화학물질 심각성, 뉴스 수집을 자동 처리
  • AI 기반 분석 — Claude AI가 매일 핵심 이슈를 분석
  • 체계적 추적 — 평가부터 개선까지 전 과정 이력 관리
